Classpublic class Sorter
InheritanceSorter flash.events.EventDispatcher

Sorter is used for sorting data in List containers. The duplicates data that is beeing assigned to it's dataProvider-property and outputs it to the sortedData-property. When added to a List container, the dataProvider property is automically populated with the dataProvider of the List, and the List display the sorted data.
