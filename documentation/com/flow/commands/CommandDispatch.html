addCompleteListener()method
public function addCompleteListener(type:Class, listener:Function, useCapture:Boolean = false, priority:int = 0, useWeakReference:Boolean = false):void

Adds a listener that is called whenever a command of a specific type has completed successfully. Only commands that don't complete instantly (e.g. RPC's or timed commands) dispatch a complete-event.

Parameters
type:Class — class of the command of whose completion you want to be notified.
 
listener:Function — function to call whenever the command has completed.
 
useCapture:Boolean (default = false) — to use capture or not.
 
priority:int (default = 0) — priority of the event handler.
 
useWeakReference:Boolean (default = false) — to use weak referencing.

addExecutedListener()method 
public function addExecutedListener(type:Class, listener:Function, useCapture:Boolean = false, priority:int = 0, useWeakReference:Boolean = false):void

Adds a listener that is called whenever a command of a specific type has been executed. This is called one frame after the command has been instantiated.

Parameters
type:Class — class of the command of whose execution you want to be notified.
 
listener:Function — function to call whenever the command has been executed.
 
useCapture:Boolean (default = false) — to use capture or not.
 
priority:int (default = 0) — priority of the event handler.
 
useWeakReference:Boolean (default = false) — to use weak referencing.
