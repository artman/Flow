Packagecom.flow.components.measuring
Classpublic class MeasureUnits
InheritanceMeasureUnits Object

A collection of measure units that can be accessed using the measeureUnits property of a Component. Accessing these is faster than to access all the measure units individually. A MeasureUnits instance is usually used in a Layout-class.

See also
