Classpublic class PaddableComponent
InheritancePaddableComponent Component flash.display.Sprite
Subclasses Label

A paddable component complements a Component with a number of properties that let the user define paddings for the contents of the component. Any fills or strokes are painted at the components original width and height, so setting padding won't affect them.

measureWithPadding()method 
protected function measureWithPadding(horizontal:Number, vertical:Number):void

Override this method to measure the dimensions of the component when extending the PaddableComponent instead of overriding the measure-method. This method is called whenever the component needs to be measured. Sub-classes should implement this to ensure that the component can be used without explicitly setting it's width and height. Implementations should measure the minimum width and height that the component requires to fully draw itself (e.g. depending on the text in a Label). The method does not return anything. Instead, for a full implementation, set the measuredWidth and measuredHeight properties of the component according to your measurement results.

Parameters

horizontal:Number — total horizontal padding to add to the measurement result.
 
vertical:Number — total vertical padding to add to the measurement result.
