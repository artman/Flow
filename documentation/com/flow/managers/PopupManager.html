public static function addPopup(popup:Component, align:String = center, effect:Effect = null, speed:Number = -1):void

Adds a new popup to the stage.

Parameters

popup:Component — popup component to add
 
align:String (default = center) — of the popup. If this parameter is PopupManager.ALIGN_CENTER, the popup's verticalCenter and horizontalCenter are set to 0, so that the popup appears in the center of the screen. Otherwise the caller is responsible for setting the popups alignment.
 
effect:Effect (default = null) — in use yet
 
speed:Number (default = -1) — speed at which to fade the popup in. If not defined, the #defaultSpeed will be used.
