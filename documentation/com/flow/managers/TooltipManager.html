Packagecom.flow.managers
Classpublic class TooltipManager
InheritanceTooltipManager Object

A singleton class to manage the display of tooltips. You usually don't call any methods of the TooltipManager directly. Instead, you set the tooltip-propert of a Component to the text you whish to display, and the Component will take care of creating and destroying the tooltip.
