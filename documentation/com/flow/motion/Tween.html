Constructor Detail
Tween()Constructor
public function Tween(target:Object = null, duration:Number = 1, values:Object = null, props:Object = null)

Constructor, commences a tween for the specified properties. Conflicting properties will be removed from any other tweens currently operating on the object. This means that you don't have to keep track which properties you tween on an object.

Parameters
target:Object (default = null) — target who's properties to tween.
 
duration:Number (default = 1) — duration of the tween in seconds.
 
values:Object (default = null) — object containing the properties and their values to tween to.
 
props:Object (default = null) — object containing additional properties for the tween. The properties are:
  • delay (Number) - How many seconds to wait before starting the tween.
  • ease (Function) - The easing equation to use
  • dontOverride (Boolean) - Whether remove any tweened property from other tweens currently operating on the object.
