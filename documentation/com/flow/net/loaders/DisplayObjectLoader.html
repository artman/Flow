Constructor Detail
DisplayObjectLoader()Constructor
public function DisplayObjectLoader(url:*, target:DisplayObjectContainer = null, replaceAllChildren:Boolean = false, context:LoaderContext = null)

Constructor. Starts loading a display object.

Parameters
url:* — URL from which to load the DisplayObject. This can be of type String or URLRequest. If String, a URLRequest is automatically created using GET.
 
target:DisplayObjectContainer (default = null) — specified, automatically adds the loaded DisplayObject as the last child of the provided DisplayObjectContainer.
 
replaceAllChildren:Boolean (default = false) — true, the loaded DisplayObject will replace all children on the target container.
 
context:LoaderContext (default = null) — loading context of the operation.
