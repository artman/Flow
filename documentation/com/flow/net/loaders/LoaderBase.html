Method Detail
addHandlers()method
public function addHandlers(completeHandler:Function, errorHandler:Function = null, statusHandler:Function = null):void

Assign functions to handle completion, errors and progress events.

Parameters

completeHandler:Function — completeHandler is invoked when the load has completed sucessfully. A single parameter is passed with this call containing the data of the load operation. The type of the data depends on the sub-class.
 
errorHandler:Function (default = null) — errorHandler is invoked when a error occurs. A single parameter is passed into the handler containing the IOErrorEvent instance.
 
statusHandler:Function (default = null) — statusHandler is invoked when the laoding progresses. A single parameter is passed into the handler containing the ProgressEvent instance.
